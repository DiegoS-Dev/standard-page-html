# Estrutura Básica de Página HTML

Este projeto demonstra a construção de uma página simples em HTML e CSS com a seguinte estrutura:

## Estrutura da Página
- **Menu (Topo)**: ocupa 10% da altura da tela e 100% da largura.
- **Conteúdo (Meio)**:
  - Coluna Esquerda: 70% da largura.
  - Coluna Direita: 30% da largura.
- **Rodapé (Base)**: ocupa 10% da altura da tela e 100% da largura.

## Características
- Todos os elementos possuem **bordas de 1px** com cores diferentes para facilitar a visualização.
- Os textos estão **alinhados à direita**.
- Os `div` estão **centralizados vertical e horizontalmente**.
- Estrutura feita com **HTML5** e estilizada com **CSS Flexbox**.

## Arquivos
- `index.html`: contém a estrutura principal da página.
- `style.css` (opcional): pode ser usado para separar os estilos do HTML.
